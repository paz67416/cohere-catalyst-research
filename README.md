# Cohere Catalyst Research

所有感（ownership）・主体感（agency）・フロー状態の事後性仮説に関する研究記録。Cohere Labs Catalyst Grant の支援を受けています。

## 研究の概要

本研究は、フロー状態や睡眠境界（入眠・覚醒の狭間）での一人称観察を出発点に、「所有感・主体感は行為に先行して成立するのではなく、事後的に構成される」という仮説（事後性仮説）を検証するものです。中核概念として、体験の成立からクオリア形成・クオリア取得へと至る三段階の媒介構造「Beral layer」を提示しています（詳細は AIP Advances 16, 045126, DOI: [10.1063/5.0325944](https://doi.org/10.1063/5.0325944)）。

将来的には、この枠組みを Human-AI Interaction および AI Alignment 研究（agency・autonomy・qualia の構造的定義の空白）へ接続することを目指しています。
